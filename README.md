# Deep Learning Models Project

## Project Overview
This project implements five neural network architectures:
1. CNN on CIFAR-10
2. FFN on MNIST
3. RNN/LSTM on IMDb
4. BERT on IMDb
5. Backpropagation on XOR

## Results Summary
| Model | Dataset | Accuracy |
|-------|---------|----------|
| CNN | CIFAR-10 | 80.22% |
| FFN | MNIST | 97.77% |
| RNN/LSTM | IMDb | 84.98% |
| BERT | IMDb | 86.40% |
| Backpropagation | XOR | 100% |
